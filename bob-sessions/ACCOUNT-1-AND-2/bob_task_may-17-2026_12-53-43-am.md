 47 | ## Layer 1: GitHub Ingestion
 48 | 
 49 | ### Purpose
 50 | Fetch and prepare repository data for analysis.
 51 | 
 52 | ### Components
 53 | 
 54 | #### `src/lib/github.ts`
 55 | 
 56 | **Key Functions:**
 57 | 
 58 | 1. **`parseRepoUrl(url: string)`**
 59 |    - Extracts owner and repo name from various URL formats
 60 |    - Supports: `https://github.com/owner/repo`, `owner/repo`
 61 |    - Returns: `GitHubRepo` object or null
 62 | 
 63 | 2. **`fetchFileTree(owner: string, repo: string)`**
 64 |    - Calls GitHub API: `/repos/{owner}/{repo}/git/trees/{branch}?recursive=1`
 65 |    - Filters for relevant files: `.js`, `.ts`, `.tsx`, `.jsx`, `.json`
 66 |    - Limits to `MAX_FILES_PER_REPO` (default: 50)
 67 |    - Tries `main` branch first, falls back to `master`
 68 | 
 69 | 3. **`fetchFileContent(owner: string, repo: string, path: string)`**
 70 |    - Fetches raw file content via GitHub Contents API
 71 |    - Decodes base64-encoded content
 72 |    - Returns UTF-8 string
 73 | 
 74 | 4. **`fetchPackageJson(owner: string, repo: string)`**
 75 |    - Special handler for `package.json`
 76 |    - Provides dependency context for analysis
 77 |    - Returns parsed JSON or null
 78 | 
 79 | 5. **`fetchRepoContext(repoUrl: string)`**
 80 |    - Orchestrates the entire fetch process
 81 |    - Returns complete `RepoContext` object
 82 |    - Includes: files, file tree, package.json, metadata
 83 | 
 84 | 6. **`buildContextString(repoContext: RepoContext)`**
 85 |    - Converts structured data into formatted string
 86 |    - Includes: repo info, dependencies, file tree, file contents
 87 |    - Optimized for Bob's consumption
 88 | 
 89 | ### Data Flow
 90 | 
 91 | ```
 92 | GitHub URL
 93 |     ↓
 94 | Parse URL → Extract owner/repo
 95 |     ↓
 96 | Fetch Tree → Get all file paths
 97 |     ↓
 98 | Filter Files → Keep relevant extensions
 99 |     ↓
100 | Fetch Contents → Download each file
101 |     ↓
102 | Build Context → Format for Bob
103 |     ↓
104 | Return RepoContext
105 | ```

    const response = await axios.post(
      `${BOB_API_URL}/analyze/stream`,
      {
        prompt: request.prompt,
        context: request.context,
        feature: request.feature,
      },
      {
        headers: getBobHeaders(),
        responseType: 'stream',
        timeout: 120000, // 2 minute timeout for streaming
      }
    );

    let fullResult = '';

    return new Promise((resolve, reject) => {
      response.data.on('data', (chunk: Buffer) => {
        const text = chunk.toString();
        fullResult += text;
        onChunk(text);
      });

      response.data.on('end', () => {
        resolve({
          result: fullResult,
          filesAnalyzed: extractFilesFromContext(request.context),
          promptUsed: request.prompt,
          timestamp: new Date().toISOString(),
          feature: request.feature,
        });
      });

      response.data.on('error', (error: Error) => {
        reject(error);
      });
    });
  } catch (error: any) {
    console.error('Bob streaming error:', error);
    throw new Error(`Error streaming from Bob API: ${error.message}`);
  }
}
