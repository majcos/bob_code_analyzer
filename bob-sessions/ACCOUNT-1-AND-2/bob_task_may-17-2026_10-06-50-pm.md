## Troubleshooting

### "GitHub API rate limit exceeded"

**Problem:** You've exceeded GitHub's API rate limit.

**Solution:**
- Verify your `GITHUB_TOKEN` is set correctly
- Authenticated requests get 5,000 requests/hour
- Wait for the rate limit to reset (shown in error message)

### "Gemini API errors"

**Problem:** Gemini API key is invalid or rate limited.

**Solution:**
- Check your `GEMINI_API_KEY` is correct
- Free tier has daily limits (resets at midnight UTC)
- Consider upgrading to a paid plan for higher limits

### "Repository not found"

**Problem:** Repository URL is incorrect or you don't have access.

**Solution:**
- Verify the repository URL is correct
- For private repos, ensure your GitHub token has `repo` scope
- Check if the repository exists and you have access

### "No analysis available"

**Problem:** Analysis hasn't been run yet or cache expired.

**Solution:**
- Click "Run Analysis" button to start analysis
- Wait for analysis to complete
- Check browser console for errors

## API Rate Limits

### GitHub API:
- **Unauthenticated:** 60 requests/hour
- **Authenticated:** 5,000 requests/hour
- **Resets:** Every hour

### Google Gemini API (Free Tier):
- **Requests:** Limited per day
- **Resets:** Daily at midnight UTC
- **Upgrade:** Available for higher limits
